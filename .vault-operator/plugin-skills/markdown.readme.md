# Markdown

Toggle between editing and reading view

## Overview

Markdown is an Obsidian core plugin. It is built into Obsidian and does not require separate installation.

## Commands

- `markdown:toggle-preview` -- Toggle reading view

## Configuration

Settings are stored at `.obsidian/markdown.json`.

To read: `read_file(".obsidian/markdown.json")`
To write: `write_file(".obsidian/markdown.json", updatedJSON)`

## Usage Notes

Plugin "Markdown" controls the note view mode.

Available commands:
- markdown:toggle-preview -- Toggle between editing mode and reading (preview) mode

Use this when the user wants to switch view modes. Also available as editor:toggle-source.
