# Random Note

Open a random note from the vault

## Overview

Random Note is an Obsidian core plugin. It is built into Obsidian and does not require separate installation.

## Commands

- `random-note:open` -- Open random note

## Configuration

Settings are stored at `.obsidian/random-note.json`.

To read: `read_file(".obsidian/random-note.json")`
To write: `write_file(".obsidian/random-note.json", updatedJSON)`

## Usage Notes

Plugin "Random Note" opens a randomly selected note.

Available commands:
- random-note:open -- Open a random note from the vault

Use this skill when the user wants to explore their vault randomly or asks for a surprise note.
