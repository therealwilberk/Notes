# Tags

Browse all tags used in the vault

## Overview

Tags is an Obsidian core plugin. It is built into Obsidian and does not require separate installation.

## Commands

- `tag-pane:open` -- Open tags pane

## Configuration

Settings are stored at `.obsidian/tag-pane.json`.

To read: `read_file(".obsidian/tag-pane.json")`
To write: `write_file(".obsidian/tag-pane.json", updatedJSON)`

## Usage Notes

Plugin "Tags" shows a browsable list of all tags used across vault notes.

Available commands:
- tag-pane:open -- Open the tags pane in the sidebar

Use this skill when the user wants to browse or explore tags. For programmatic tag searching, prefer the search_by_tag tool.
