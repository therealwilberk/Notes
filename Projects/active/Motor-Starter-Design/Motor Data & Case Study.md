# Motor Data & Case Study

The established motor used across all three configurations.

## Parameters

| Parameter | Value |
|---|---|
| Power | 7.5 kW |
| Voltage | 230Δ / 400Y |
| Full load current (FLC) | 15.9A (star, 400V supply) |
| Supply | 3-phase, 400V / 230V, 50Hz, TN-S |
| Speed | 1455 RPM (4-pole) |
| Insulation class | F |
| Duty | S1 continuous |
| Application | Centrifugal pump (low starting torque — suits star-delta) |

## Protection Basis

All device ratings, cable sizes, and protection settings are calculated and justified against these motor parameters. Every selection must be correct enough that a site electrician could wire from the drawings.
